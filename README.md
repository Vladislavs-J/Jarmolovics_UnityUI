# Jarmolovics_UnityUI - Tēla apģērbšanas spēle

Šis projekts izstrādāts ieskaites ietvaros, pielietojot Unity UI elementus (text, image, button, toggle, slider, scroll view, dropdown, input field). 
Spēlē lietotājs var izvēlēties tēlu, ievadīt tā vārdu un dzimšanas gadu, kā arī apģērbt to, izmantojot drag and drop.
